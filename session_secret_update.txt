SESSION SECRET FIX

Replace:
app/main.py
with the file from this folder.
Default value
If SESSION_SECRET is not configured, the application uses:

development-only-change-this-secret

Change it in PowerShell
Run this before starting the server:
$env:SESSION_SECRET="your-long-random-secret"
python -m uvicorn app.main:app --reload --reload-dir app


Important
The PowerShell value works only in the current terminal window.
After changing SESSION_SECRET, existing login cookies become invalid.
Users need to log in again.
